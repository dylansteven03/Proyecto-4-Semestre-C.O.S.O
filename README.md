# Proyecto-4-Semestre-C.O.S.O
Aviaciones el C.O.S.O es una plataforma web orientada a la gestión de servicios de una aerolínea. Permite a los usuarios registrarse, iniciar sesión, consultar vuelos disponibles, acceder a ofertas y conocer convenios con hoteles. Además, cuenta con un panel administrativo para gestionar información del sistema, usuarios y servicios.
